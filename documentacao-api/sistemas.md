# Sistemas

{% hint style="success" %}
Vale lembrar que nosso sistema **não** se limita a fazer integrações somente com as plataformas informadas abaixo.
{% endhint %}

{% hint style="info" %}
Dependendo da necessidade do cliente ou do formato da integração, será necessário um [Sistema Integrador](https://ajuda.simplesdesk.com.br/documentacao-api/sistemas-integradores) para tratar dados e melhorar a performance.
{% endhint %}
