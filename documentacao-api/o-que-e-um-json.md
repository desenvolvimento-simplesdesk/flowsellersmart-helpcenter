# O que é um JSON?

**JSON (JavaScript Object Notation)** é um formato simples de troca de dados. Imagine que é uma maneira de organizar informações de forma clara, usando pares de "nome" e "valor". Isso permite que computadores e sistemas diferentes entendam e compartilhem dados entre si.

Por exemplo, pense em uma lista de compras:

* Nome: "Maçã"
* Quantidade: "10"

Em JSON, isso seria escrito assim:

```json
{
  "Nome": "Maçã",
  "Quantidade": 10
}
```

Basicamente, é como fazer uma lista de informações que podem ser facilmente lidas e usadas por programas de computador.
