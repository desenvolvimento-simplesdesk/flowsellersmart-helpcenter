# API de usuários (atendentes)

Esta API será utilizada para obter todos os usuários do sistema e seus respectivos Id's. Os id's vão ser importantes quando você precisar forçar um ticket para abrir em usuário, utilizando a [API (PUSH)](https://ajuda.simplesdesk.com.br/documentacao-api/api-de-templates).

Este processo será o mesmo modelo da [API de templates](https://ajuda.simplesdesk.com.br/documentacao-api/api-de-templates), porém ao invés de adicionar o "**templates"** na url, você adicionar "**users".**
